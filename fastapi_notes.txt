FastApi Commands
1. pip3 install "fastapi[all]"
    -> To install the FastApi
2.  pip3 freeze
    -> To list out the installed packages
3. uvicorn main:app --reload or uvicorn app.main:app --reload
    -> first app is package folder
    -> main is main.py file name
    -> second app is fastapi instance in main.py
    -> To run the server
4. pip3 install psycopg2
    -> Used to connect with postgres database
5. pip3 install psycopg2
    -> to upgrade pip
